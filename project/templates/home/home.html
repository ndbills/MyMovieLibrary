Welcome to 'My Movie Library'

Whether your movies are in the movie cabinet or loaned to a friend, this easy-to-use web application can help you track your ever-growing movie library.

Get Started, it's fast, easy and free!

Once you have created an account, start making libraries and adding your movies to those libraries.

On-loan, keep track of that movie!

When you want your movie back, you can send an email asking for the movie to be returned.

Open Source, give back to the community!

This project is open source, so make suggestions, contribute or even fork our project on github!

Responsive Design

This web application was made to be responsive, so you can use this on any device!
